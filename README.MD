# Vodyani ark

🛸  Vodyani ark 是一个完备的配置管理解决方案。

通过在项目中引入 ark，开发者可以轻易的管理本地的配置数据，并且根据配置的变化，动态地根据配置重建本地客户端（如 MySQL/Redis 等）🤩

此外，ark 还提供了远程配置中心的数据同步和数据订阅功能 🎉
